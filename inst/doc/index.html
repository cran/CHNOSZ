Vignettes of package CHNOSZ

If you're just starting to use this package, you might want to read the CHNOSZ-package help file first.

abundance.pdf:
Calculating relative abundances of proteins
pie.pdf:
Pie charts for protein abundances
eqdata.pdf:
Extract data from EQ6 output files
check.csv:
Species that fail self-consistency check of thermodynamic parameters (generated using info().)
